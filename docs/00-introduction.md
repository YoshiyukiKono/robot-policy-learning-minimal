# 00 Introduction — このリポジトリで学ぶこと

このリポジトリは、ロボット／Embodied Agent 学習の最小ループを体験するための教材です。

最小ループとは次の流れです。

```text
observation -> policy -> action -> reward -> update
```

CartPole を題材に、Gymnasium の環境を動かし、Stable-Baselines3 の PPO で方策を学習し、評価し、最後に動画として確認します。

## 目的

この教材の目的は、高性能なロボット制御モデルを作ることではありません。

目的は、次の問いに自分で答えられるようになることです。

- 環境とは何か
- 観測とは何か
- 方策とは何か
- 行動とは何か
- 報酬とは何か
- PPO は何を更新しているのか
- 学習と評価は何が違うのか
- 動画生成では何が起きているのか

## このリポジトリが意図的に避けているもの

最初の一歩として、次のものは扱いません。

- ROS 2
- 実ロボット制御
- Isaac Sim / Isaac Lab
- Vision-Language-Action model
- Diffusion Policy
- 分散学習

これらは重要ですが、最初に触ると「何が本質なのか」が見えにくくなります。

## 最初に理解すべきこと

ロボットポリシー学習は、単純化すると次のような構造です。

```text
世界の状態を観測する
↓
方策が行動を選ぶ
↓
環境が次の状態と報酬を返す
↓
その経験を使って方策を改善する
```

CartPole はロボットアームではありません。しかし、制御対象があり、観測があり、行動があり、報酬があるという点で、ロボット学習の最小模型として非常に有用です。
